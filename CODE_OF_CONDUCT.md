# 行为准则

## 我们的承诺

我们致力于为所有参与者提供一个无骚扰、尊重且友好的协作环境。

## 建议行为

有助于建设社区的行为包括：

- 尊重且建设性地沟通。
- 理性接受反馈并持续改进。
- 优先考虑社区整体利益。

不可接受的行为包括：

- 人身攻击、侮辱、歧视或骚扰。
- 公开或私下的威胁、恐吓。
- 未经允许公开他人隐私信息。

## 执行

维护者负责解释并执行本准则，并可对不当行为采取必要措施。
